# WoW Tools OTA Data

Data files for the WoW Gear Optimizer, served via GitHub Pages.

Files:
- `version.json` — Current data version with SHA-256 checksums
- `item_drops.json` — Item drop source database
- `talent_builds.json` — Meta talent builds per spec

Base URL: `https://kevibes.github.io/wow-tools-data/`
